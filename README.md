# 🛍️ SHOPPING-SITE

A **responsive shopping website** built using **HTML, CSS, and JavaScript**.  
This project showcases a multi-page frontend ecommerce site with product browsing, cart page, and user interaction features.

## 🧱 Features

✅ Home page showcasing products  
✅ Shop / Category pages (Men, Women, Accessories, Sale)  
✅ Product detail page  
✅ Shopping cart page  
✅ User login & registration forms  
✅ Contact & About pages  
✅ Responsive layout for desktop and mobile

# 🛍️ SHOPPING-SITE

A responsive **frontend shopping website** built using **HTML, CSS, and JavaScript**.  
This project demonstrates a multi-page e-commerce UI with product listings, cart page, and user authentication pages.

---

## 📌 Project Overview

The SHOPPING-SITE is a static e-commerce website that allows users to:
- Browse products by category
- View product details
- Add products to cart (UI level)
- Access login and registration pages

This project focuses on **frontend design and responsiveness**.

---

## 🚀 Features

- 🏠 Home page with featured products  
- 🛒 Shop & category pages (Men, Women, Accessories, Sale)  
- 📄 Product detail page  
- 🧺 Shopping cart page  
- 🔐 Login & Register pages  
- ℹ️ About, Contact & Blog pages  
- 📱 Fully responsive design

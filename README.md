Shopping Website
This is a fully responsive, multi-page shopping website built using HTML, CSS, and JavaScript. It provides a smooth and modern shopping experience with a clean and user-friendly interface.

Responsive Design
The website is fully responsive and works well on mobile devices, tablets, and desktops. It is built using modern CSS layout techniques such as Flexbox and Grid.

Multiple Pages
The project includes multiple pages such as Home, Products, Product Details, Cart, and optional pages like Contact or About.

Functional Add to Cart
Users can add products to the cart dynamically. They can increase or decrease product quantities, remove items, and view real-time updates of the total cart price.

Clean and Modern UI
The interface is simple, neat, and visually appealing. It includes smooth hover effects and well-structured layouts for a better user experience.

Pure Frontend
The website is built completely using HTML, CSS, and vanilla JavaScript without any frameworks, making it lightweight, fast, and easy to customize.
